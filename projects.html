Interactive map of Södermalm

Since moving to Stockholm, I have been doing illustrations of people and places around the city's Södermalm neighborhood. I wanted to display a few of these images on a map indicating their location, so I created an SVG map with markers for each location, and added some interactivity with JavaScript - hovering over the marker displays the location name, and clicking on the marker shows the illustration associated with that place.
